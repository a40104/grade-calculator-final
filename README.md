# 🎓 학점 계산기 Final Term Project

> **- Java 프로젝트 제출용**

---

## 📌 프로젝트 소개

이 프로젝트는 사용자가 입력한 과목별 학점과 성적을 기반으로  
**총 GPA(평균 평점)** 를 계산하는 Java 기반 콘솔 애플리케이션입니다.

---

## 🧩 주요 기능

- 📚 과목명, 학점 수, 성적 등급 입력
- 🔄 성적 → 평점 자동 변환 (A+ → 4.5 등)
- 📊 총 GPA 계산
- ❗ 잘못된 입력 감지 (예: 존재하지 않는 성적 등급)
- ✅ JUnit 기반 자동 테스트 포함

---
